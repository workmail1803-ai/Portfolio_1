# Nafis Hossain Momen — Portfolio Website

A premium, modern, responsive developer portfolio website combining glassmorphism card designs, bento grid showcases, and elegant scroll animations.

## 🚀 Features

- **Hero Section**: Canvas-based interactive particle field, floating parallax shapes, dynamic typed title, and interactive stats counter.
- **About (Bento Grid)**: Clean grid showcasing biography, tech stack tools, education, and coding philosophy.
- **Services**: Glassmorphism cards with modern neon hover-glow border effects.
- **Projects**: Clean filtering system (All, Web, Mobile, AI/ML) linked directly to GitHub repositories.
- **Contact Form**: High-end glassmorphism card form with floating labels and visual loading states.

## 🛠️ Built With

- **HTML5** & **CSS3** (Vanilla styling, variables, CSS Grid, animations)
- **Vanilla JavaScript** (Intersection Observer, custom particles, typewriter, state handlers)

## 📦 How to Deploy

### 1. GitHub Pages (Recommended & Free)
1. Go to your repository on GitHub: `https://github.com/workmail1803-ai/Portfolio_1`
2. Click on **Settings** tab.
3. On the left sidebar, click on **Pages**.
4. Under **Build and deployment** -> **Source**, select **Deploy from a branch**.
5. Under **Branch**, select `main` and `/ (root)`, then click **Save**.
6. Wait 1-2 minutes, and your site will be live at `https://workmail1803-ai.github.io/Portfolio_1/`!

### 2. Vercel (Free)
1. Go to [vercel.com](https://vercel.com) and sign in with GitHub.
2. Click **Add New** -> **Project**.
3. Import your `Portfolio_1` repository.
4. Click **Deploy**.
